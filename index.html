⚙️ Configurar Firebase (tempo real)
Passo a passo (5 minutos):
1. Acesse console.firebase.google.com
2. Clique em Adicionar projeto → nome: zat-tirolez-2026
3. No menu lateral: Build → Firestore Database → Criar banco → Modo produção
4. No menu lateral: Authentication → Sign-in method → Email/senha → Ativar
5. Authentication → Users → Adicionar usuário: admin@zat.tirolez.com.br + sua senha
6. Configurações do projeto → Seus apps → Adicionar app Web → Copie as credenciais abaixo
Cole as credenciais do Firebase e salve o arquivo HTML — o sistema conectará automaticamente.
